# エンジニア副業ラボ

公開URL: https://nyx1271.github.io/money-system/

エンジニア向けに、副業ブログ・個人開発・学習環境・ASP活用を扱う静的サイトです。GitHub Pagesで公開しています。

## 収益方針

収益の中心は、読者の行動に自然につながるアフィリエイト導線です。

優先順位:

1. A8.netなどのASP登録・案件確認
2. ドメイン・サイト公開準備
3. 学習用品・作業環境
4. 個人開発・ブログ運営に関係するサービス

広告を先に押し出すのではなく、読者の悩みを解決したあとに「必要な人だけ確認する」形で設置します。

## 記事作成ルール

自動生成記事でも、次の要素を必ず入れます。

- SEOタイトル: 検索キーワード + 読者メリット + 具体性
- メタディスクリプション: 80〜120文字程度
- 冒頭の答え: 読者がすぐ結論を分かるようにする
- 具体例: エンジニア・副業・ブログ運営に寄せる
- 失敗例: 広告の貼りすぎ、テーマが広すぎる、次の行動がない等
- CTA: 記事内容に合う1つだけを優先
- 内部リンク: 次に読む記事を2〜3本

## 自動生成の品質基準

生成記事は、同じ本文を使い回さないこと。

最低限、以下を記事ごとに変える。

- 読者の悩み
- 結論
- 手順
- 失敗例
- CTA文
- 関連記事

詳細は `content-quality-logic.md` を参照。
