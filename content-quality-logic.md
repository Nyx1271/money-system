# Content Quality Logic

## Goal
Improve auto-generated articles so they are useful and actionable.

## Core Rules

Each article must include:

- Clear user problem
- Specific conclusion (actionable)
- 3-5 step process
- At least 2 common mistakes
- 1 strong call-to-action
- 2-3 internal links

## Title Rule

Format:
Keyword + Benefit + Specificity

Example:
SEO → SEO beginner first 3 steps
Python → Python side income simple project guide

## Structure

1. Answer first
2. Reason
3. Steps
4. Mistakes
5. CTA

## Avoid

- Reusing same sentences
- Abstract advice only
- No concrete examples

## Minimum Check

- Unique content
- Clear action for reader
- Relevant CTA
